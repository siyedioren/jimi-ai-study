## 题目描述

实现Array类的>>、 <<的运算符重载。已知Array类的定义（不允许增加其它成员函数）及main中的代码如下（不允许修改）：

class Array{

public:

       Array(); //初始化length=0

       void Insert(int i, double x);  //在下标i处插入x

    friend ostream &operator<<(ostream &out,Array &arr);  //<<运算符重载

       friend istream &operator>>(istream &in,Array &arr);    //>>运算符重载

private:

       double data[MaxSize]; //存储元素，MaxSize为常量，需先定义

       int length;              //顺序表实际长度

};

int main(){

    Array a;

       cin>>a;

       cout<<a;

       return 0;

}

请完成Array类中的成员函数，使之能够通过运算符重载实现数组中数据的输入和输出。数据的输入格式为：第一个数是数组中的数据个数，之后为数组中的数据值。

5 2.3 5.6 7.98 6.5 6.32
