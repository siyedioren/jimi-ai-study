# 1105: 类与对象11：Complex类1

题目链接: http://59.73.87.16/problem.php?id=1105

## 题目描述

定义一个复数类Complex，该类对象存放一个复数的实部和虚部。设计带有默认形参值的构造函数，实部虚部的默认值均为0；设计一个能够输出复数的displayComplex成员函数（当虚部为0时仅输出实部；当实部为0时仅输出虚部。当虚部为1或-1时，虚部只输出+i或-i。输出样式如：18.5、10+3i、10-3.5i、-5.8+9.5i、-4.5-9.6i、0、4-i、-6+i等）。

main函数如下(不得修改main函数)：




int main()





{





       double r1,i1;





       cin>>r1>>i1;





       Complex c1(r1,i1);





       c1.displayComplex();





       return 0;


}

## 样例输入 
								Copy

2.5 -3.8

## 样例输出								Copy

2.5-3.8i
