# 1111: 类与对象17：Point类5

题目链接: http://59.73.87.16/problem.php?id=1111

## 题目描述

定义一个点类Point，数据成员为平面点坐标x，y，成员函数有：构造函数，函数displayPoint显示点信息，成员函数calDistance实现计算两个点间距离。函数原型double calDistance(const Point &p)。主函数中输入两个点信息，定义两个类对象，利用displayPoint显示各自的点信息，再利用calDistance计算两点距离。

## 样例输入 
								Copy

0 0 10 20

## 样例输出								Copy

Point(0,0)
Point(10,20)
Distance:22.3607
