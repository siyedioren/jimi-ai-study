# 1097: 类与对象03：Point类3

题目链接: http://59.73.87.16/problem.php?id=1097

## 题目描述

定义一个Point类，数据成员包括私有数据成员为double类型的点坐标x，y；成员函数包括构造函数Point（用于实现对数据成员x，y的初始化）、成员函数setPoint（用于改变数据成员x、y的值）、成员函数leftMove（点坐标向左移动delta_x）、成员函数上移upMove（点坐标上移delta_y）、成员函数displayPoint（用于输出点坐标x、y，输出格式为点坐标用逗号分隔并半角圆括号括起来）。

main函数如下(不得修改main函数)：

int main()

{

       double x1,y1,x2,y2,dx,dy;

       cin>>x1>>y1;

       cin>>dx;

       cin>>dy;

       cin>>x2>>y2;

       Point p1;

       p1.displayPoint();

       p1.setPoint(x1,y1);

       p1.displayPoint();

       p1.LeftMove(dx);

       p1.displayPoint();

       p1.UpMove(dy);

       p1.displayPoint();

       Point p2(x2,y2);

       p2.displayPoint();

       return 0;

}

## 样例输入 
								Copy

10 25.5 5 10 5.5 20

## 样例输出								Copy

(0,0)
(10,25.5)
(5,25.5)
(5,35.5)
(5.5,20)
