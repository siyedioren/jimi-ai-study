# 1096: 类与对象02：Point类2

题目链接: http://59.73.87.16/problem.php?id=1096

## 题目描述

定义一个Point类，数据成员包括私有数据成员为double类型的点坐标x，y；成员函数包括构造函数Point（用于实现对数据成员x，y的初始化）、成员函数setPoint（用于改变数据成员x、y的值）、成员函数displayPoint（用于输出点坐标x、y，输出格式为点坐标用逗号分隔并半角圆括号括起来）。

main函数如下(不得修改main函数)：

int main()

{

       double x1,y1,x2,y2;

       cin>>x1>>y1;

       cin>>x2>>y2;

       Point p1(x1,y1);

       p1.displayPoint();

       p1.setPoint(x2,y2);

       p1.displayPoint();

       return 0;

}

## 样例输入 
								Copy

10 25.5 5.5 20

## 样例输出								Copy

(10,25.5)
(5.5,20)
