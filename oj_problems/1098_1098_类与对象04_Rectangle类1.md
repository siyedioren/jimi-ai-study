# 1098: 类与对象04：Rectangle类1

题目链接: http://59.73.87.16/problem.php?id=1098

## 题目描述

定义一个长方形类Rectangle，私有数据成员为double型width、height（表示长方形的宽和高），成员函数包括构造函数Rectangle（用于实现对数据成员width、height的初始化）、成员函数getArea（计算并返回长方形的面积）。

main函数如下（不得修改main函数）：

int main()

{

       double width,height;

       cin>>width>>height;

       Rectangle r1;

       cout<<r1.getArea()<<endl;

       Rectangle r2(width,height);

       cout<<r2.getArea()<<endl;

       return 0;

}

## 样例输入 
								Copy

10.2 25.5

## 样例输出								Copy

0
260.1
