# 基米编程学习器 — 优化计划

> 由 Kimi Work + Kimi Code 联合分析生成

---

## 📊 项目现状

**技术栈**: Next.js 15 + React 19 + TypeScript + Sass
**核心功能**: AI 编程辅导聊天界面（当前仅支持 C++ 基础模式）
**数据存储**: 纯 localStorage（会话、题目）
**AI 后端**: Moonshot API 代理

**当前问题**:
- `ChatInterface.tsx` 单文件 613 行，职责过重
- 首页只有一道硬编码题目，模式选择隐藏在聊天页下拉框里
- AI 回复纯文本显示，无代码高亮/ Markdown 渲染
- `ModeCard` 组件已写好但从未使用
- 无流式响应、无错误重试、无设置面板

---

## 🎯 优化计划（按优先级）

### 🔴 Phase 1 — 核心体验（P0，预计 1 周）

| 任务 | 说明 | 影响 |
|------|------|------|
| **1.1 拆分 ChatInterface** | 拆分为 `SessionSidebar`、`MessageList`、`ChatInput`、`ModeSelector` + `useSessions`/`useChat` hooks | 可维护性提升，为后续功能铺路 |
| **1.2 Markdown + 代码高亮** | 引入 `react-markdown` + `react-syntax-highlighter`，替换纯文本渲染 | AI 回复可读性大幅提升 |
| **1.3 API 流式响应** | `/api/chat` 改用 `ReadableStream`，前端逐字显示 | 长回复不再卡顿，体验接近 ChatGPT |
| **1.4 模式持久化** | 将当前选中的模式存入 localStorage，刷新后保留 | 避免每次重选 |
| **1.5 清理死代码** | 决定 ModeCard 去留，删除 `test.html` | 减少技术债务 |

### 🟡 Phase 2 — 功能扩展（P1，预计 1 周）

| 任务 | 说明 | 影响 |
|------|------|------|
| **2.1 首页模式网格** | 用现有 `ModeCard` 在首页展示所有学习模式（C++ 基础 / C++ ACM / 通用逻辑） | 用户一眼看到全部功能 |
| **2.2 补全 persona.json** | 为 `cpp_acm` 和 `general` 模式写完整的 systemPrompt 和 API 参数 | 三个模式都能正常使用 |
| **2.3 题目库系统** | 创建 `data/problems.json`，首页每日一题从中动态读取 | 可扩展多题目，支持按模式筛选 |
| **2.4 错误重试机制** | API 失败时显示重试按钮，指数退避 | 网络波动不再中断学习 |
| **2.5 文本可复制** | 移除全局 `user-select: none`，代码块加复制按钮 | 用户可保存 AI 讲解 |
| **2.6 设置面板** | 齿轮按钮弹出设置：主题切换、字体大小、清空会话、API Key 覆盖 | 基础自定义能力 |

### 🟢 Phase 3 — 体验增强（P2，预计 1 周）

| 任务 | 说明 | 影响 |
|------|------|------|
| **3.1 学习进度追踪** | localStorage 记录完成题目数、连续学习天数、各模式统计 |  gamification，提升留存 |
| **3.2 深色模式** | CSS 变量切换 + `data-theme` 属性，持久化偏好 | 夜间学习更舒适 |
| **3.3 移动端适配** | 侧边栏可折叠（<768px 汉堡菜单），底部会话抽屉 | 手机可用 |
| **3.4 键盘快捷键** | `Ctrl+K` 新建会话、`Ctrl+[/]` 切换会话、`Esc` 取消输入 | 效率提升 |
| **3.5 导出对话** | 支持导出为 Markdown / JSON | 方便整理笔记 |

### ChatInterface 拆分（Phase 1.1）

```
components/
├── Chat/
│   ├── ChatInterface.tsx      #  orchestrator（<100 行）
│   ├── SessionSidebar.tsx     #  会话列表 + 新建/删除
│   ├── MessageList.tsx        #  消息渲染（Markdown + 高亮）
│   ├── MessageBubble.tsx      #  单条消息（用户/AI 样式）
│   ├── ChatInput.tsx          #  输入框 + 发送 + 模式选择
│   ├── ModeSelector.tsx       #  模式下拉（从 persona.json 读取）
│   └── TypingIndicator.tsx    #  AI 输入中动画
├── hooks/
│   ├── useSessions.ts         #  localStorage 会话 CRUD
│   ├── useChat.ts             #  发送消息 + 流式接收
│   └── useAutoScroll.ts       #  自动滚动到底部
```
